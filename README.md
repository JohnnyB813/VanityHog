# VanityHog
A Vanity address generator for warthog addresses


**Usage:**
`python3 vanity.py --prefix=00000` - Remember to only search for hex characters 0-9 and a-f.

**Requirements:** python3
**pip libraries:** pycryptodome v3.5.0, pycoin, ecdsa
